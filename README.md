# GitHub Knowledge Graph MCP Server

MCP server for exploring and analyzing GitHub repository code structures.

## Features

- Semantic code analysis for GitHub repositories
- Code structure and relationship exploration
- Component, function, and class search
- MCP interface for language model integration

## Installation

```bash
npm install
npm run build
```

## Usage

```bash
npm start
```

## Available Tools

- `analyze_repository`: Analyze a GitHub repository and create a knowledge graph
- `explore_graph`: Explore relationships in the knowledge graph
- `search_nodes`: Search for nodes in the knowledge graph
- `get_node_details`: Get detailed information about a specific node
- `find_dependencies`: Find dependencies of a node
- `semantic_search`: Perform meaning-based search in the codebase
